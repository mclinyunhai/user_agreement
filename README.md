## 信息收集和保护：

我们十分重视用户的隐私保护。

### 我们收集的信息内容：

访问凌云海官方网站时：

- 您访问时的 IP
- HTTP 请求摘要内容
- 请求信息，包括发起请求的时间等

在凌云海 QQ 交流群交流时：

- 您的聊天记录

### 收集信息用途：

我们不会在未经用户许可的情况下透露任何用户敏感信息。这至少包括：

- 访问网站日志
- QQ 聊天记录
- 用户信息

**特别声明：** 当中华人民共和国政府部门、国家机关要求我们披露的相关信息时，以上无效。
